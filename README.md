# Travel Advisor

Travel Advisor is a React app for location based discovery of restaurants, hotels, and attractions.

## Product problem

Travel planning requires fast filtering, clear cards, map context, and useful external data. This project focuses on the frontend side of that workflow: search, browse, compare, and open details without burying the user.

## What it shows

| Area | Evidence |
| --- | --- |
| API connected UI | External place data, map based browsing, cards, ratings, and location oriented views. |
| Frontend product flow | Search, filters, lists, map context, and responsive layout choices. |
| React structure | Component based interface, state driven rendering, and Material UI usage. |
| Hospitality fit | The app is about helping people decide where to go, what to eat, and what to explore. |

## Stack

React, JavaScript, Material UI, Maps APIs, CSS.

## Run locally

```bash
npm install
npm start
```

Build for production.

```bash
npm run build
```
